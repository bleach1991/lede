---------------------------------------------------
**主要提供x86_64及R2S，R4S固件**

**固件主要分为高大全版(plus)和精简版(mini)**

**文件下载请看文件夹名称下载**

**如文件夹名中：\x86_64\2021-04-03__05-03-19--plus-daily表示x86_64高大全版本，编译时间为2021年4月3日**

**固件每日凌晨4点自动重新编译发布，保留历史版本，下载最新版本即可**
